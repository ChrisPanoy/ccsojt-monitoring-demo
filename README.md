# OJT Monitoring System - Static Demo

This is a high-fidelity static version of the OJT Monitoring System, designed for demonstration purposes and hosting on GitHub Pages.

## Features
- **High-Fidelity UI/UX**: Matches the production system's styling, layout, and animations.
- **Role-Based Access**: Demo authentication for Admin, Student, and Coordinator roles.
- **Pure Static**: Authored in Vanilla HTML, CSS, and JavaScript. No backend required.
- **Mocked Functionality**: Forms and tables exhibit realistic behavior using local state.

## Demo Credentials
Access the various dashboards using the following credentials:
- **Admin**: `admin@demo.com`
- **Student**: `student@demo.com`
- **Coordinator**: `teacher@demo.com`
- Any password will work.

## Deployment to GitHub Pages
1. Push the contents of this folder (`static-demo`) to a new GitHub repository.
2. Go to Repository **Settings** > **Pages**.
3. Under **Build and deployment**, set the source to **Deploy from a branch**.
4. Select the `main` (or `master`) branch and the `root` folder.
5. Save and wait for the deployment to finish.

## Tech Stack
- **Framework**: Bootstrap 4 & RuangAdmin Template
- **Styling**: Vanilla CSS3 & Tailwind CSS (Student Dashboard)
- **Logic**: Vanilla JavaScript (ES6)
- **Icons**: FontAwesome 5
- **Data Display**: jQuery DataTables
